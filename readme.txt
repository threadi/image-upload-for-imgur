Upload images in Block Editor directly to imgur.

== Description ==

Upload images in Block Editor directly to imgur. You need API credentials from imgur to use this plugin.

---

== Installation ==

1. Upload "imgur-image-upload" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Enter your imgur API credentials in the settings.
4. Use the Block "Imgur Image Upload".

== Frequently Asked Questions ==

= Can I use the plugin without a imgur account? =

The plugin can be installed even without Imgur account. However, it is not usable without imgur API credentials.
