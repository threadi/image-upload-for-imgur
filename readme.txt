Upload images in Block Editor directly to Imgur.

== Description ==

Upload images in Block Editor directly to Imgur. You need [API credentials](https://api.imgur.com/oauth2/addclient) from Imgur to use this plugin.

== Hints ==

This plugin is not part of Imgur's services. It was developed independently to make the good service of Imgur usable in WordPress. Questions about Imgur itself should be directed to their support: [https://help.imgur.com](https://help.imgur.com)

The Imgur logo as part of all distributed icons is a trademark of [MediaLab.Ai Inc.](https://imgur.com/tos).

== Repository ==

The development repository is on [GitHub](https://github.com/threadi/image-upload-for-imgur).

== Installation ==

1. Upload "image-upload-for-imgur" to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Enter your imgur API credentials in the settings.
4. Use the Block "Image Upload via Imgur".

== Frequently Asked Questions ==

= Can I use the plugin without an Imgur account? =

The plugin can be installed even without Imgur account. However, it is not usable without Imgur API credentials.
